=== Quick Product Image Upload ===
Tags: woocommerce, product, image upload, product image
Requires at least: 3.9
Tested up to: 3.9

Easily upload and delete product images on main product screen, without having to edit each product. 

== Description ==

Great for product image management in WooCommerce!
This plugin adds a column to the product listing in WooCommerce with links to upload and delete product images.
Uses AJAX - the page does not refresh and there is no need to edit each product individually.
Just click the "Set Image" link and find or upload an image.
Changes are instantly reflected on front-end - no need to press "update" button.
Saves a tremendous of time!

== Installation ==


1. Upload th zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Products" to view the new column added where you can upload and delete images for all the products listed.
